# Intake Summary Report - 设计文档

## 📋 需求分析

### 产品定位
**Intake Summary Report** — 企业内部项目需求管理仪表板，用于监控项目进度、费用及 SLA 风险状态。

### 目标用户
- **主要用户**: 项目经理、部门经理、交付团队
- **使用场景**: 日常项目监控、周报汇报、资源调配决策

### 核心功能需求

| 功能模块 | 需求描述 | 优先级 |
|---------|---------|--------|
| **部门筛选** | 按部门(EC/Marketing/Retail/SCM/MP&A/Corporate/Tech)快速筛选 | P0 |
| **KPI 概览** | 实时统计 Intake 总数、未开始、进行中、已关闭、已取消 | P0 |
| **数据可视化** | 状态占比饼图 + 任务类型分布柱状图 | P1 |
| **数据表格** | 展示 Intake 详情，支持搜索、排序 | P0 |
| **时间筛选** | 按日期范围过滤数据 | P1 |
| **导出功能** | 导出 Excel 报表 | P2 |
| **SLA 预警** | Pending 天数超过阈值时高亮显示(>14天红色, >7天黄色) | P1 |

### 数据字段
- Intake 号码、部门、提交人、创建日期
- 任务类型(开发任务/配置任务/项目任务/额外资金申请任务)
- Pending 天数、状态、费用、批复人、具体内容、关联 BRD

## 🎨 UI 设计方案

### 设计理念
**"简洁专业，突出内容"** — 采用 lululemon 品牌风格，以白色为基底，红色作为视觉标识，打造适合技术/交付团队的专业仪表板。

### 配色方案

| 用途 | 颜色 | 色值 |
|-----|------|------|
| 主背景 | 纯白 | `#FFFFFF` |
| 页面背景 | 浅灰 | `#F8FAFC` |
| 品牌标识 | lululemon 红 | `#E31937` |
| 主文字 | 深 slate | `#1E293B` |
| 次要文字 | slate-500 | `#64748B` |
| 边框 | slate-200 | `#E2E8F0` |

### 状态颜色

| 状态 | 颜色 | 背景 |
|-----|------|------|
| 未开始(Open) | 蓝色 | `#DBEAFE` / `#2563EB` |
| 进行中(In Progress) | 琥珀色 | `#FEF3C7` / `#D97706` |
| 已关闭(Closed) | 翠绿色 | `#D1FAE5` / `#059669` |
| 已取消(Cancelled) | 灰色 | `#F1F5F9` / `#475569` |
| SLA 严重 | 红色 | `#FEE2E2` / `#DC2626` |
| SLA 警告 | 琥珀色 | `#FEF3C7` / `#D97706` |

### 交互设计

1. **KPI 卡片点击**: 点击卡片筛选对应状态的数据
2. **图表折叠**: 点击标题栏可展开/折叠图表区域
3. **悬停效果**: 表格行悬停高亮，KPI 卡片悬停上浮
4. **实时搜索**: 输入时即时过滤表格数据
5. **日期筛选**: 选择日期范围后自动更新数据

## 📐 原型设计说明

### 组件清单

| 组件 | 说明 |
|-----|------|
| Header | 标题区 + 搜索框 + 导出按钮 |
| DeptFilter | 部门筛选按钮组 |
| KPICards | 5个 KPI 统计卡片 |
| ChartSection | 可折叠的图表区域 |
| DateFilter | 日期范围选择器 |
| DataTable | 数据表格 + 分页 |

### 响应式断点

- **Desktop**: > 1024px — 完整布局
- **Tablet**: 768px - 1024px — 调整间距
- **Mobile**: < 768px — 堆叠布局

### 动画规范

| 动画 | 时长 | 缓动函数 |
|-----|------|---------|
| KPI 卡片悬停 | 250ms | cubic-bezier(0.4, 0, 0.2, 1) |
| 图表展开/折叠 | 300ms | ease-out |
| 按钮悬停 | 200ms | ease |
| 表格行悬停 | 150ms | ease |
