# Intake Summary Dashboard - 需求分析文档

## 1. 项目概述

### 1.1 产品定位
**Intake Summary Dashboard** 是一个项目需求管理仪表盘，用于：
- 实时监控项目进度
- 跟踪费用支出
- 预警 SLA 风险
- 支持多部门协作

### 1.2 目标用户
- 项目经理
- 部门负责人
- 财务审批人员
- 交付团队

---

## 2. 功能需求分析

### 2.1 核心功能模块

| 模块 | 功能描述 | 优先级 |
|------|---------|--------|
| **数据概览** | KPI 卡片展示总数、各状态数量 | P0 |
| **部门筛选** | 按部门（EC/Marketing/Retail等）筛选 | P0 |
| **数据表格** | 展示 Intake 详细信息列表 | P0 |
| **搜索功能** | 按号码/提交人搜索 | P0 |
| **图表统计** | 状态分布、类型统计可视化 | P1 |
| **分页功能** | 每页20条，支持页码跳转 | P0 |
| **导出功能** | 导出 Excel 报表 | P1 |
| **时间筛选** | 按日期范围筛选 | P2 |

### 2.2 数据字段

| 字段 | 说明 | 示例 |
|------|------|------|
| Intake 号码 | 唯一标识 | INT-2024001 |
| 部门 | 所属部门 | EC/Marketing/Retail |
| 提交人 | 创建人 | 张三 |
| 类型 | 任务类型 | 开发任务/配置任务/项目任务 |
| Pending 天数 | 待处理天数 | 12 天（含 SLA 预警） |
| 状态 | 当前状态 | 未开始/进行中/已关闭/已取消 |
| 费用 | 项目费用 | ¥45,000 / TBD |
| 批复人 | 审批人 | Manager A |
| Scope | 项目范围 | 电商页面改版 |
| 关联项 | 关联 BRD | BRD-992 |

### 2.3 费用显示规则

| 状态 | 费用显示 | 说明 |
|------|---------|------|
| Open | **TBD** | 待确定，虚线边框 |
| In Progress | **TBD** | 待确定，虚线边框 |
| Closed | **¥xx,xxx** | 实际费用 |
| Cancelled | **-** | 横线表示无费用 |

### 2.4 SLA 预警规则

| Pending 天数 | 显示样式 | 风险等级 |
|-------------|---------|---------|
| ≤14 天 | 普通文字 | 正常 |
| 15-30 天 | 黄色背景 | 警告 |
| >30 天 | 红色背景 | 危险 |

---

## 3. 交互需求

### 3.1 部门筛选交互
- 点击部门按钮，表格数据实时过滤
- 当前选中部门高亮显示
- 支持 "All" 查看全部

### 3.2 图表交互
- 点击展开/收起按钮切换显示
- 图表随筛选条件动态更新
- 显示各状态数量和百分比

### 3.3 分页交互
- 点击页码跳转
- 上一页/下一页按钮
- 当前页码高亮

### 3.4 搜索交互
- 实时搜索（防抖）
- 支持 Intake 号码和提交人

---

## 4. 技术需求

### 4.1 前端技术栈
- **框架**: 原生 HTML + JavaScript
- **样式**: Tailwind CSS
- **图表**: Chart.js
- **图标**: Font Awesome

### 4.2 响应式设计
- 桌面端: 完整布局
- 平板: 自适应网格
- 移动端: 单列布局，横向滚动表格

### 4.3 性能要求
- 首屏加载 < 2s
- 筛选响应 < 200ms
- 支持 1000+ 条数据流畅渲染

---

## 5. 设计规范

### 5.1 色彩系统

| 用途 | 颜色 | 色值 |
|------|------|------|
| 主色调 | 蓝色 | #3b82f6 |
| 成功/已关闭 | 翠绿 | #10b981 |
| 警告/进行中 | 琥珀 | #f59e0b |
| 危险/SLA超期 | 红色 | #ef4444 |
| 取消/禁用 | 灰色 | #9ca3af |
| 背景 | 浅灰 | #f8fafc |
| 卡片背景 | 白色 | rgba(255,255,255,0.7) |

### 5.2 字体规范
- **主字体**: Inter, system-ui
- **标题**: 24px bold
- **正文**: 14px regular
- **辅助文字**: 12px
- **数据数字**: 30px black

### 5.3 间距规范
- 卡片内边距: 20px (p-5)
- 卡片间距: 16px (gap-4)
- 圆角: 12px (rounded-xl)
- 表格行高: 64px

### 5.4 阴影规范
- 卡片阴影: 0 1px 3px rgba(0,0,0,0.05)
- 悬浮阴影: 0 8px 30px rgba(0,0,0,0.08)

---

## 6. 页面结构

```
Intake Summary Dashboard
├── Header (标题 + 搜索 + 导出)
├── Department Filters (部门筛选)
├── KPI Cards (5个指标卡片)
├── Collapsible Charts (可折叠图表区)
│   ├── 状态分布饼图
│   └── 类型统计图
├── Filter Bar (时间筛选 + 记录数)
├── Data Table (数据表格)
│   ├── Table Header
│   ├── Table Body (20行)
│   └── Table Footer (Summary)
└── Pagination (分页控件)
```
