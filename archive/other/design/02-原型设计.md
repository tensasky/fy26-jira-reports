## 2. 组件原型详解

### 2.1 Header 区域

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│   Intake Summary Dashboard                                      │
│   实时监控项目进度、费用及 SLA 风险状态                          │
│                                                                 │
│                                    ┌─────────────┐ ┌─────────┐ │
│                                    │ 🔍 搜索...  │ │ 📥 导出 │ │
│                                    └─────────────┘ └─────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**交互说明:**
- 搜索框实时搜索，防抖 300ms
- 导出按钮点击后生成 Excel 文件下载

---

### 2.2 部门筛选器

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│  [All]  [EC]  [Marketing]  [Retail]  [SCM]  [MP&A]  [Corp] [Tech]│
│   ↑                                                      
│  选中状态（蓝色背景）                                            │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**交互说明:**
- 单选模式
- 选中项蓝色高亮
- 点击后表格数据实时过滤

### 2.3 KPI Cards

```
┌─────────────┐  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐
│             │  │  ░░░░░░░░░  │  │  ░░░░░░░░░  │  │  ░░░░░░░░░  │  │  ░░░░░░░░░  │
│  Intake 总数 │  │   未开始    │  │   进行中    │  │   已关闭    │  │   已取消    │
│             │  │  ░░░░░░░░░  │  │  ░░░░░░░░░  │  │  ░░░░░░░░░  │  │  ░░░░░░░░░  │
│     60      │  │     15      │  │     18      │  │     18      │  │      9      │
│             │  │   (蓝色)    │  │   (琥珀色)   │  │   (翠绿色)   │  │   (灰色)    │
└─────────────┘  └─────────────┘  └─────────────┘  └─────────────┘  └─────────────┘

```

**设计说明:**
- 卡片顶部 3px 渐变条，颜色对应状态
- 数字使用 30px black 字体
- 悬浮时轻微上浮 + 阴影增强

**状态徽章样式:**
- 未开始: 蓝色背景 + 蓝色文字
- 进行中: 琥珀背景 + 琥珀文字
- 已关闭: 翠绿背景 + 翠绿文字
- 已取消: 灰色背景 + 灰色文字

**SLA 预警样式:**
- 正常: 普通文字
- 警告 (>14天): 黄色背景高亮
- 危险 (>30天): 红色背景高亮

## 3. 响应式布局

### 3.1 桌面端 (≥1024px)
- 5列 KPI Cards
- 完整表格显示
- 图表并排显示

### 3.2 平板端 (768px-1024px)
- 3列 KPI Cards
- 表格横向滚动
- 图表堆叠显示

### 3.3 移动端 (<768px)
- 2列 KPI Cards
- 表格卡片化或横向滚动
- 图表单列显示
- 筛选器横向滚动

---

## 4. 交互动效

### 4.1 页面加载
- KPI 数字从 0 动画增长到实际值
- 卡片依次淡入（stagger 100ms）

### 4.2 筛选交互
- 按钮点击时轻微缩放（scale 0.95）
- 表格数据淡入淡出切换

### 4.3 悬浮效果
- 卡片悬浮上移 2px + 阴影增强
- 表格行悬浮背景变色
- 按钮悬浮亮度增加

### 4.4 图表动画
- 展开时高度从 0 过渡到 auto
- 饼图数据更新时旋转动画
