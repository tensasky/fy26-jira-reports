# Intake Summary Dashboard - UI 设计规范

## 1. 设计原则

### 1.1 设计目标
- **清晰**: 信息层级分明，一目了然
- **高效**: 快速定位关键数据，减少认知负担
- **专业**: 符合企业级应用审美，建立信任感
- **响应式**: 适配多种设备，保持一致体验

### 1.2 设计灵感
- **Glassmorphism**: 玻璃态卡片，现代感强
- **数据可视化**: 图表直观展示数据分布
- **lululemon 风格**: 简约白色背景 + 红色竖线标识

## 2. 色彩系统

### 2.1 主色调

## 3. 字体规范

### 3.1 字体家族
```css
font-family: 'Inter', system-ui, -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif;
```

### 3.2 字体层级

| 层级 | 大小 | 字重 | 用途 |
|------|------|------|------|
| H1 | 24px | 700 (bold) | 页面标题 |
| H2 | 18px | 600 (semibold) | 区块标题 |
| H3 | 16px | 600 (semibold) | 卡片标题 |
| Body | 14px | 400 (regular) | 正文内容 |
| Small | 12px | 400 (regular) | 辅助文字、标签 |
| Caption | 10px | 600 (semibold) | KPI标签、表头 |
| Data | 30px | 900 (black) | KPI数字 |

### 3.3 行高
- 标题: 1.2
- 正文: 1.5
- 紧凑: 1.25

## 10. 动效规范

### 10.1 过渡时间
- 快速: 150ms (按钮点击)
- 正常: 200ms (悬浮效果)
- 慢速: 300ms (展开/收起)

### 10.2 缓动函数
- 默认: `ease`
- 悬浮: `ease-out`
- 展开: `ease-out`

### 10.3 动画效果
```css
/* 淡入 */
@keyframes fadeIn {
  from { opacity: 0; transform: translateY(10px); }
  to { opacity: 1; transform: translateY(0); }
}

/* 卡片悬浮 */
.card:hover {
  transform: translateY(-2px);
  transition: transform 0.3s ease, box-shadow 0.3s ease;
}

/* 展开动画 */
.collapsible {
  max-height: 0;
  overflow: hidden;
  transition: max-height 0.3s ease-out;
}
.collapsible.expanded {
  max-height: 500px;
}
```

## 12. 设计检查清单

### 12.1 视觉检查
- [ ] 色彩对比度符合 WCAG 2.1 AA 标准
- [ ] 文字清晰可读（最小 12px）
- [ ] 按钮尺寸适合点击（最小 44x44px）
- [ ] 阴影使用一致

### 12.2 交互检查
- [ ] 所有可点击元素有悬浮状态
- [ ] 加载状态有反馈
- [ ] 错误状态有提示
- [ ] 动画流畅不卡顿

### 12.3 响应式检查
- [ ] 移动端布局正确
- [ ] 表格可横向滚动
- [ ] 按钮大小适合触摸
- [ ] 文字不溢出
